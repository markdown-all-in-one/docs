See Markdown syntax and flavors for details.

Looking for other syntax? Markdown All in One is known to be compatible with many other extensions. The following have been tested:

  • Mermaid Supportopen in new window
  • Markdown Emojiopen in new window
  • Markdown Footnotesopen in new window
  • Markdown Superscriptopen in new window
  • Markdown+Mathopen in new window (when markdown.extension.math.enabled is false)
